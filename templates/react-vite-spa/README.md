# React Vite SPA

This template is for React SPA, Vite, and PWA-style applications.

It includes a minimal Vite app, AGENTS workflow, `lean-context`, roadmap,
snapshots, technical debt tracking, and GitHub workflow guidance.

## Requirements

- Node.js 22 or newer.
- npm 11 or newer.

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Verification

```bash
npm run build
```

## Version

The authoritative version source is `package.json`.

## AGENTS CLI

This template includes `@mvuljevas/agents` as optional project tooling.

```bash
npm run agents:doctor
npm run agents:setup
npm run agents:dashboard
```

Use `agents:dev` only when the template defines an application dev command.
