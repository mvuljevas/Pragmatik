# React Vite SPA Shell

This template is an Pragmatik shell for a future React, Vite, SPA, or PWA project.

It intentionally contains no `package.json`, HTML, source files, Vite config, or
runtime dependencies. Define the application goal first, then let Pragmatik help
create the correct project files.

## Start

```bash
pragmatik doctor
pragmatik setup --dry-run
pragmatik suggest --idea "Describe the React application"
```
