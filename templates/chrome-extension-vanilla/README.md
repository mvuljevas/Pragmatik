# Chrome Extension Vanilla Shell

This template is an Pragmatik shell for a future Chrome extension using vanilla
HTML, CSS, and JavaScript.

It intentionally contains no `manifest.json`, background script, content script,
popup, options page, or runtime files. Define the extension goal first, then let
Pragmatik helps create the correct extension files.

## Start

```bash
pragmatik doctor
pragmatik setup --dry-run
pragmatik suggest --idea "Describe the Chrome extension"
```
