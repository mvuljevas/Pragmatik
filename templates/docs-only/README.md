# Docs Only Shell

This template is an Pragmatik shell for documentation, workflow, governance, or
planning repositories.

It intentionally contains no generated project content beyond Pragmatik workflow
files. Define the project goal first, then let Pragmatik help populate the real
documentation structure.

## Start

```bash
pragmatik doctor
pragmatik setup --dry-run
pragmatik suggest --idea "Describe the documentation project"
```
