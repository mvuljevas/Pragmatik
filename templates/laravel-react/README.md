# Laravel React Shell

This template is an Pragmatik shell for a future Laravel application with a
React-capable frontend.

It intentionally contains no Laravel, Composer, Node, Vite, Inertia, or React
runtime files. Define the application goal first, then let Pragmatik help create
the correct stack files.

## Start

```bash
pragmatik doctor
pragmatik setup --dry-run
pragmatik suggest --idea "Describe the Laravel application"
```
